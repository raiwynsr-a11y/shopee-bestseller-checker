# Shopee Best-Seller Checker (Unofficial)

โปรแกรมเดสก์ท็อปสำหรับดึง "สินค้าขายดี" จากการค้นหาใน Shopee แล้วบันทึกเป็น Excel/CSV  
> ใช้ endpoint สาธารณะที่เว็บไซต์ Shopee เรียกใช้งาน (ไม่เป็นทางการ) ซึ่งอาจเปลี่ยนเมื่อใดก็ได้

## ฟีเจอร์
- โหมดค้นหา 2 แบบ: **Keyword** หรือ **ทั้งร้าน (Shop ID/URL)**
- ฟิลเตอร์: **ช่วงราคา**, **ยอดขายขั้นต่ำ**, **Category ID/URL**
- **Batch Keywords**: ดึงได้หลายคีย์เวิร์ดในรอบเดียว พร้อมคอลัมน์บอกที่มา
- ดึงชื่อร้านค้าอัตโนมัติ
- ปุ่ม **หยุด** ระหว่างดึงข้อมูล
- เปิดลิงก์สินค้าได้จากตาราง
- Export เป็น **Excel/CSV**

## วิธีติดตั้งและใช้งาน (Windows, ไม่ใช้คำสั่ง)
1) ติดตั้ง **Python 3.10 หรือ 3.11** จาก https://www.python.org/ (กด Next ต่อไปจนจบ)
2) ดาวน์โหลดโปรเจกต์จาก GitHub: ปุ่ม **Code → Download ZIP** แล้วแตกไฟล์
3) ดับเบิลคลิก `build.bat` เพื่อ:
   - ติดตั้งไลบรารีที่ต้องใช้อัตโนมัติ
   - ติดตั้ง PyInstaller
   - สร้างไฟล์ `dist/ShopeeBestSeller.exe`
4) เปิด `dist/ShopeeBestSeller.exe` และใช้งาน:
   - เลือกโดเมนประเทศ
   - ใส่คีย์เวิร์ด หรือ Shop ID/URL
   - ตั้งหน้าสูงสุด/ยอดขายขั้นต่ำ/ช่วงราคา/Category
   - กด **ดึงข้อมูล** → ดูผล → **บันทึกเป็น Excel/CSV**

## หมายเหตุ
- ใช้เพื่อการวิเคราะห์ส่วนบุคคล ปฏิบัติตามเงื่อนไขการใช้บริการของ Shopee เสมอ
- หากเรียกใช้งานถี่อาจถูกจำกัด (429) — โปรแกรมมี backoff ให้แล้ว
- ราคาใน API เป็นหน่วยย่อย โปรแกรมคำนวณให้เรียบร้อย

## ทำตัวติดตั้ง (.exe Installer)
1) สร้าง `dist/ShopeeBestSeller.exe` ตามขั้นตอนด้านบน
2) ติดตั้ง **Inno Setup**
3) เปิด `installer.iss` ใน Inno Setup → กด **Build**
4) จะได้ไฟล์ `dist_installer/ShopeeBestSeller_Setup.exe`
